## 📌 Project Overview
In today’s digital world, knowing how to use technology is just as important as knowing how to use it responsibly. This project was designed to help me understand both aspects.  

As part of this assignment, I worked as a *Student Digital Ambassador*, where I explored different platforms, created awareness content, and learned how to guide others in using digital tools safely and effectively.  

This project is divided into five tasks, each focusing on a different part of digital literacy — from building a professional profile to understanding cyber safety.

---

## 🎯 Objectives
- Understand the importance of digital literacy  
- Build a professional online presence  
- Learn proper email etiquette and communication  
- Explore coding and collaboration platforms  
- Create awareness about cyber safety and phishing
